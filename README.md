## NZBN verification using TypeScript

## Introduction
    NZBN verification in node js. nzbn verification using TypeScript. We have provide nzbn verification api.

## Setup Steps:
`npm install`
### local server
`npm run start:dev`
### prod build
`npm run build`
### prod build run
`node dist/index.js`
